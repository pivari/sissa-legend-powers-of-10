Technical & Scientific Documentation v1.3

Welcome to the documentation for Sissa's Legend & Powers of 10. This interactive resource explores exponential growth and its application to the physical world.

1. Mathematical Foundations

The journey follows a geometric progression: \( a_n = 2^{n-1} \). While the start seems trivial, the total number of grains by the 64th square is astronomical.

Total Grains \( S_{64} = \sum_{i=0}^{63} 2^i = 2^{64} - 1 \)

Result: 18,446,744,073,709,551,615 grains.

2. Physical Scale Milestones

To provide intuition, we map these numbers to real-world objects:

  • Square 9 (~1,000 Grains): A handful.
  • Square 19 (~1,000,000 Grains): One full grain sack (~45kg).
  • Square 29 (~1,000,000,000 Grains): A large commercial grain truck.
  • Square 59 (~1 Quintillion): More than the stars in our galaxy.

3. New in Version 1.3

3.1 Internationalization (i18n) & Default Language

The application now defaults to English to ensure global accessibility. Language preferences are stored locally via localStorage, allowing the app to remember the user's choice across sessions.

3.2 Enhanced PWA Metadata

The manifest.json has been updated with full support for screenshots in both narrow (mobile) and wide (desktop) formats, improving the installation experience in modern browsers.

4. Technical Implementation

  • BigInt Precision: Uses JavaScript BigInt to accurately handle the massive integers required for the 64th square without losing precision.
  • Multilingual Engine: UI strings and story segments are dynamically loaded from external JSON files (en.json, it.json).
  • Offline Capability: A Service Worker (sw.js) caches essential assets, including the new language files and icons, ensuring full offline functionality.
