Internationalization Project

We invite students and professors worldwide to help us translate this experiment. Education should have no language barriers.

How to Contribute

1. Fork the project on GitHub.

2. Open languages.js and add your translation like this example:

'fr': {
    voiceCode: 'fr-FR',
    story: {
        s1: "La légende raconte che le Brahmane Sissa...",
        s2: "Le Roi fut surpris par cette demande...",
        // ... continue with s3, s4, s5
    },
    ui: {
        btnNext: "Suivant",
        btnStart: "Commencer l'aventure",
        // ... follow the existing structure
    }
}

Collaboration Context

Before translating, please consider the scientific pillars of the project:

  • The Legend: On the 64th square of the board, the grains are \(2^{63}\). The total sum is \(2^{64}-1\).
  • The Science: Starting from 5mm, we halve the size 63 times (\(2^{-63}\)) to reach the subatomic scale.

Professors can use this as a classroom exercise for both Science and Computer Science students.
